# Basic Projects with HTML, CSS and JavaScript

This repository contains a collection of **basic projects developed with HTML, CSS, and JavaScript**, focused on strengthening the fundamentals of frontend web development through practical and functional exercises.

The main goal is to **learn by doing**, applying essential concepts of structure, styling, and JavaScript logic, as well as good coding and organization practices.

---

## Repository Objectives

- Strengthen the fundamentals of **HTML5, CSS3, and JavaScript**
- Understand the interaction between structure, styles, and behavior
- Develop small functional projects from scratch
- Serve as an **introductory frontend development portfolio**

---

Each project can be run by opening the `index.html` file directly in a web browser.

---

## Technologies Used

- **HTML5** – Semantic structure
- **CSS3** – Styling and layout
- **JavaScript (ES6)** – Logic, events, and DOM manipulation
